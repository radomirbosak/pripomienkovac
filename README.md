Pripomienkovac
==============

An older java project used to store notes, search through them and remind future events. Programmed in [Netbeans IDE](https://netbeans.org/).

The `main` function is located in the [`src/rado/pripomienkovac/RunClass.java`](https://github.com/radomirbosak/pripomienkovac/blob/master/src/rado/pripomienkovac/RunClass.java) file.

Uses: GUI (swing), MySQL
